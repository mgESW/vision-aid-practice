# Vision Aid Practice Site
 
This site Vision-Aid constituents to upload operational data to a database. 

Automating data entry will allow the service providers to have a seamless data storage and entry for resources provided and other operational attributes.

## What is Nextra?

A **simple**, **powerful** and **flexible** site generation framework with everything you love from Next.js.
